# Spin（飞刀大师）— 需求摘要

## 1. 产品定位

第一人称视角的 3D 物理投掷游戏：玩家蓄力投掷飞刀，与气球、场景与布娃娃人体交互；视觉为描边/墨水感风格（白底、自定义光照方向）。

## 2. 核心玩法

### 机制

- **扎入**：飞行的 **轨迹与入射角合适** 时，**刀刃** 可扎在 **墙、桌子、小人** 等物体上（扎住后随可动物体运动或留在静态表面上）。
- **气球**：被 **刀刃** 扎中则 **爆炸**（表现上爆裂并销毁，计入关卡「扎破气球」数量）。
- **掠过风压**：飞刀从气球 **附近掠过**（未扎中）时产生 **风**，使气球 **轻微晃动**。

| 能力 | 要求 |
|------|------|
| 视角 | 鼠标捕获时有限偏航/俯仰；`Esc` 切换捕获/释放；点击非捕获区可重新捕获 |
| 投掷 | 按住左键蓄力（有力度条 UI），松手沿相机朝向前方抛出，速度在最小～最大间随蓄力插值；飞刀带自旋与轻微上抛弧线 |

## 3. 关卡设计

胜利条件均为：**扎破规定数量的气球**（计数以气球被戳破销毁为准）。三关共用投掷与物理规则。

| 关卡 | 场景要点 | 胜利条件 |
|------|----------|----------|
| **第一关** | 静态布置（如桌面等） | 扎破 **3** 个气球 |
| **第二关** | 小人会 **躲避** 飞刀 | 扎破 **6** 个气球 |
| **第三关** | **Wheel 旋转**；小人与气球 **绑在 Wheel 上**，小人 **不躲避** | 扎破 **6** 个气球 |

场景资源可对应：`Chapter1` / `Chapter2` / `Chapter3`（以实际摆阵与流程为准）。

## 4. 技术约束（与实现一致）

- 引擎：Godot 4.x，3D 物理为 **Jolt**；碰撞层含平台、布娃娃、静态、人体受击检测等分工。
- 关卡需能统计本关气球被戳破数量，并在达标时触发胜利（具体 UI、下一关衔接可后续补充）。
